5. Macros y VBA

Macro

Una macro es un pequeño programa informático que ejecuta una acción o un conjunto de acciones tras su activación. Las macros se escriben con el lenguaje de programación VBA (Visual Basic for Applications) en módulos. Para usar una macro, se necesitan bastantes conocimientos de VBA.

Esta es una tabla dinámica bidimensional donde se pueden examinar determinadas columnas y filas.

El usuario puede agrupar datos en categorías, filtrar datos para incluir o excluir categorías y crear gráficos con tablas dinámicas.

  • Registro de ventas de un proyecto

VBA

Una macro es un pequeño programa informático que ejecuta una acción o un conjunto de acciones tras su activación. Las macros se escriben con el lenguaje de programación VBA (Visual Basic for Applications) en módulos. Para usar una macro, se necesitan bastantes conocimientos de VBA.

Cuando el usuario haga clic en “Show High Sales” (Mostrar las ventas más grandes) escrito en AVBA, Excel calculará las ventas más grandes.

Las macros de VBA permiten automatizar tareas. Sin embargo, debido a su complejidad, muchos usuarios de Excel no saben usarlas.

Los usuarios avanzados de Excel utilizan macros de VBA para lo siguiente:

  • Organizar y formatear datos con rapidez.
  • Ver datos en todas las hojas.
  • Automatizar propiedades y acciones en tablas dinámicas.
  • Crear y modificar formularios de usuarios.
  • Diseñar sistemas para crear, actualizar o modificar archivos de Excel.
