3. Formato condicional avanzado

¿De qué se trata?

El formato condicional es ideal para diseñar hojas de cálculo, ya que el usuario puede añadir con rapidez una capa de análisis visual en un conjunto de datos. También se puede usar para crear herramientas avanzadas, como mapas de calor y bolas de Harvey.

Un mapa de calor es una representación de datos donde los valores se representan con colores. Por lo general, se usa un espectro de colores cálidos a fríos para indicar que un valor es más grande o pequeño que otro.

En este ejemplo, se indica que las temperaturas de Central Park son más altas entre mayo y septiembre.
