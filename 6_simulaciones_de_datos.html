6. Simulaciones de datos

¿De qué se trata?

Por lo general, una simulación implica la creación de un modelo matemático para representar las características de un sistema. Si bien existen programas informáticos especializados para las simulaciones modernas, los usuarios con habilidades avanzadas de Excel pueden usar las herramientas del programa para aprovechar esta función.


Ejemplo de simulación de datos: proyección de ganancias futuras en función de datos históricos.

Para hacer una simulación en Excel, el usuario necesita conocer algunas funciones, como las siguientes:

  • Funciones para la generación de números aleatorios
  • Funciones de estadística
  • Tablas de datos
  • Herramientas de análisis de hipótesis
  • Estos son algunos casos de uso posibles de las simulaciones de datos:
  • Simular actividades de publicidad en línea para determinar las posibles ventas promedio en un período.
  • Encontrar patrones de ventas en función de datos recientes
  • Modelar el valor del tiempo de vida del cliente a partir de su historial de compras de productos.
